# Assets Folder

This folder contains example images and resources for the application.

## Structure

```
assets/
├── examples/          # Example images for users to try
├── icons/            # Application icons and branding
└── README.md         # This file
```

## Example Images

Add sample images here for users to test the application without uploading their own.

Recommended image formats:
- JPG/JPEG
- PNG
- BMP

## Usage

Place example images in the `examples/` subdirectory. The application will automatically detect and offer them to users.
