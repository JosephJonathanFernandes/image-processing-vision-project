The Image Processing Vision Project is designed to make computer vision and image processing accessible through an intuitive web interface. This documentation provides detailed information about using, contributing to, and understanding the project.

## FAQ

### General Questions

**Q: What image formats are supported?**
A: JPG, JPEG, and PNG formats are currently supported.

**Q: Is there a file size limit?**
A: The default limit is 200MB, but this can be configured.

**Q: Can I use this commercially?**
A: Yes, this project is licensed under MIT, which allows commercial use.

### Technical Questions

**Q: What Python version is required?**
A: Python 3.8 or higher is required.

**Q: Can I run this without internet?**
A: Yes, once installed, the application runs completely offline.

**Q: How do I report a bug?**
A: Use our [bug report template](.github/ISSUE_TEMPLATE/bug_report.md) to open an issue.
